# 🎓 Online Scholarship Management System (OSMS)

A full-stack MERN application for managing scholarships with role-based access for students and admins.

## 🌐 Live Demo

🔗 https://osms-scholarship-system.vercel.app/

## 🚀 Features
- JWT Authentication (Student/Admin)
- OTP Login System
- Scholarship Application Workflow
- Document Upload & Verification
- Cloudinary File Storage
- Admin Dashboard

## 🛠 Tech Stack
- MongoDB, Express.js, React.js, Node.js
- Tailwind CSS
- Cloudinary, Multer
- Nodemailer

## ⚙️ Setup

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
